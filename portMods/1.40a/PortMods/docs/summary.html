Processing PortMods library

Purpose

Built for Processing 3, PortMods are pieces of Java code with input and output ports that can be chained. This can help you draw more complex animations, like patterns and shapes.

For a detailed description, see https://commonpike.github.io/nl.kw.processing.mods/docs/PortMods.pdf
